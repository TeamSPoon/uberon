
## Class : magnocellular part of red nucleus

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0002936
 * *Label* = magnocellular part of red nucleus
 * *Definition* = The magnocellular red nucleus (RNm) is located in the rostral midbrain and is involved in motor coordination. Together with the parvocellular red nucleus, it makes up the red nucleus. [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + http://en.wikipedia.org/wiki/Magnocellular_red_nucleus]
