
## Class : gyrus rectus

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0004671
 * *Label* = gyrus rectus
 * *Definition* = The portion of the frontal lobe medial to the medial orbital gyrus is named the gyrus rectus (or straight gyrus), and is continuous with the superior frontal gyrus on the medial surface. A specific function for the gyrus rectus has not yet been elucidated.
