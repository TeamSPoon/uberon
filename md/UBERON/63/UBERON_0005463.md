
## Class : subcapsular sinus of lymph node

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0005463
 * *Label* = subcapsular sinus of lymph node
 * *Definition* = A space between the capsule and the cortex which allows the free movement of lymphatic fluid and so contains a sparsity of lymphocytes. It is continuous with the similar lymph sinuses that flank the trabeculae. [ [database_cross_reference](../../ef/oboInOwl#hasDbXref.md) = https://en.wikipedia.org/wiki/Lymph_node#Subcapsular_sinus ]
