
## Class : intercalary element of fore digit

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_3000856
 * *Label* = intercalary element of fore digit
 * *Definition* = Small elements present in all digits of the manus, when present in a taxon. Each intercalary element articulates with the anteroventral end of the penultimate phalanx proximally and the distal phalanx distally.
