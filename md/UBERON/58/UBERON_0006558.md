
## Class : lymphatic part of lymphoid system

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0006558
 * *Label* = lymphatic part of lymphoid system
 * *Definition* = An organ system subdivision that is a network of vessels capable of removing accumulating protein and fluid from the interstitial space and returning it to the vascular space. In some species, this network is connected to the immune system via lymph nodes and lymphocyte-producing organs, with the whole being the lymphoid system.
