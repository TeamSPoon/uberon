
## Class : ventricular system of central nervous system

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0005281
 * *Label* = ventricular system of central nervous system
 * *Definition* = a set of structures containing cerebrospinal fluid in the brain. It is continuous with the central canal of the spinal cord[WP]. [ [database_cross_reference](../../ef/oboInOwl#hasDbXref.md) = Wikipedia:Ventricular_system ]
