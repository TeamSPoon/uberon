
## Class : inferior salivatory nucleus

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0002872
 * *Label* = inferior salivatory nucleus
 * *Definition* = In the brain, the inferior salivatory nucleus is a cluster of neurons controlling the parasympathetic input to the parotid gland. It is one of the components of the glossopharyngeal nerve. [WP,unvetted]. [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + Wikipedia:Inferior_salivatory_nucleus]
