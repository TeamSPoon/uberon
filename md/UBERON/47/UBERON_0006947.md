
## Class : male genital duct

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0006947
 * *Label* = male genital duct
 * *Definition* = A duct or series of ducts that transports sperm from the gonad. In mammals this is from the seminiferous tubules through rete testis, vas efferentia, epididymis, vas deferens, ejeculatory duct to the urethra. [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + https://orcid.org/0000-0002-6601-2165]
 * *Comment* = Taxon notes: we use this as a generic grouping class for a number of taxa. AO notes: in FMA includes as parts: seminal vesicle, efferent ductule (vas efferentia), epididymis, deferent duct (vas deferens), ejaculatory duct
