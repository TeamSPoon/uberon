
## Class : red nucleus

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0001947
 * *Label* = red nucleus
 * *Definition* = The red nucleus is a structure in the rostral midbrain involved in motor coordination. It comprises a caudal magnocellular and a rostral parvocellular part. [WP,unvetted]. [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + Wikipedia:Red_nucleus]
 * *Comment* = Function notes: in animals without a significant corticospinal tract, gait is mainly controlled by the red nucleus.
