
## Class : nigrostriatal tract

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0014169
 * *Label* = nigrostriatal tract
 * *Definition* = The term nigrostriatal fibers refers to a dopaminergic fiber pathway connecting the substantia nigra with the striatum. It is not readily distinguished in myelin-stained cross-sections (Carpenter-83). [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + NIF_GrossAnatomy:birnlex_4107]
