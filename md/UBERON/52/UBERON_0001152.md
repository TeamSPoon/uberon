
## Class : cystic duct

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0001152
 * *Label* = cystic duct
 * *Definition* = The cystic duct is the short duct that joins the gall bladder to the common bile duct. It usually lies next to the cystic artery. It is of variable length. It contains a 'spiral valve', which does not provide much resistance to the flow of bile. [WP,unvetted]. [ [database_cross_reference](../../ef/oboInOwl#hasDbXref.md) = Wikipedia:Cystic_duct ]
