
## Class : intercalary element of hind digit

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_3000931
 * *Label* = intercalary element of hind digit
 * *Definition* = Small elements present in all digits of the pes, when present in a taxon. Each intercalary element articulates with the anteroventral end of the penultimate phalanx proximally and the distal phalanx distally.
