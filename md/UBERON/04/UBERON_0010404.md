
## Class : lateral ventricle subependymal layer

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0010404
 * *Label* = lateral ventricle subependymal layer
 * *Definition* = A cell layer below the ependyma in the lateral ventricles of the brain. This region contains adult neural stem cells which have the potential to generate new neurons and glial cells via neurogenesis. It is an adult version of the embryonic forebrain germinal zone [[database_cross_reference](../../ef/oboInOwl#hasDbXref.md) =  + Wikipedia:Subependymal_zone]
