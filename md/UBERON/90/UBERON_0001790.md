
## Class : outer plexiform layer of retina

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0001790
 * *Label* = outer plexiform layer of retina
 * *Definition* = The layer within the retina where the bipolar cells synapse with the photoreceptor cells[ZFA]. Between the inner and outer nuclear layers, the outer plexiform layer (OPL) contains connections between the photoreceptors and bipolar and horizontal cells[GO].
