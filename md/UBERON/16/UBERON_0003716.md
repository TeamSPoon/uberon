
## Class : recurrent laryngeal nerve

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0003716
 * *Label* = recurrent laryngeal nerve
 * *Definition* = A branch of the vagus nerve that supplies motor function and sensation to the larynx (voice box). It travels within the endoneurium. It is the nerve of the 6th Branchial Arch. [ [database_cross_reference](../../ef/oboInOwl#hasDbXref.md) = Wikipedia:Recurrent_laryngeal_nerve ]
