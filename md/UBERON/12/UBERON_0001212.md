
## Class : duodenal gland

 * *IRI* = http://purl.obolibrary.org/obo/UBERON_0001212
 * *Label* = duodenal gland
 * *Definition* = A compound tubular submucosal gland found in that portion of the duodenum which is above the hepatopancreatic sphincter (Sphincter of Oddi). The main function of these glands is to produce a mucus-rich alkaline secretion (containing bicarbonate)[WP].
