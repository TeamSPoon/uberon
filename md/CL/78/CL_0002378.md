
## Class : immature Vgamma2-positive fetal thymocyte

 * *IRI* = http://purl.obolibrary.org/obo/CL_0002378
 * *Label* = immature Vgamma2-positive fetal thymocyte
 * *Definition* = A double negative thymocyte that has a T cell receptor consisting of a gamma chain containing a Vgamma2 segment, and a delta chain. This cell type is CD4- negative, CD8-negative and CD24-positive and is found in the fetal thymus.
