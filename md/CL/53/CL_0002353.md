
## Class : fetal liver hematopoietic progenitor cell

 * *IRI* = http://purl.obolibrary.org/obo/CL_0002353
 * *Label* = fetal liver hematopoietic progenitor cell
 * *Definition* = A hematopoietic stem cell that resides in the fetal liver. In mice, this cell type is first observed at E10.5. This cell type is MHC-positive, HSA-positive, AA4.1-positive, CD45-positive, Sca-1 positive, CD150-positive, CD48-negative and CD244-negative.
