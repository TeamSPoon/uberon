
## Class : type L enteroendocrine cell

 * *IRI* = http://purl.obolibrary.org/obo/CL_0002279
 * *Label* = type L enteroendocrine cell
 * *Definition* = A enteroendocrine cell type that is numerous in ileum, present in jejunum and large intestine, few in duodenum. This cell type produces glucagon-like immunoreactants (glicentin, glucagon-37, glucagon-29, GLP-1 and -2) and PYY.
