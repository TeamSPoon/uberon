
## Class : unswitched memory B cell

 * *IRI* = http://purl.obolibrary.org/obo/CL_0000970
 * *Label* = unswitched memory B cell
 * *Definition* = An unswitched memory B cell is a memory B cell that has the phenotype IgM-positive, IgD-positive, CD27-positive, CD138-negative, IgG-negative, IgE-negative, and IgA-negative.
